## How to setup?
Only windows testing
##### Install
For python

`pip install flask`
`pip install flask-cors`

For vue (vite)

`cd picinity-front`
`npm install`
##### Run
Backend

`set FLASK_DEBUG=1`
`python.exe main.py `

Front

`npm run dev`
